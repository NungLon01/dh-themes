# Midnight Nebula Theme

A deep-space inspired theme with glowing gradients, glassmorphism, and starfields.

## Key Visuals
*   **Background**: Radial gradients simulating distant stars on a #05070a void.
*   **Glass**: Cards use `backdrop-filter: blur(16px)` for a HUD-like feel.
*   **Neon**: Primary action color is a Purple-Pink gradient (`--primary`, `--accent`).
*   **Interaction**: Elements glow and lift on hover.

## Typography
*   Font: `Outfit` (Google Fonts) - Geometric and modern sans-serif.

## Install
Copy `style.css` and use class structure from `demo.html`.
