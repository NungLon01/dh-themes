# Paper Sketch Theme

A hand-drawn, playful UI theme mimicking a sketch on paper.
Perfect for creative portfolios, indie games, or casual tools.

## Key Features
*   **Hand-drawn Borders**: Uses irregular `border-radius` to simulate pencil strokes.
*   **Wiggle Animation**: Cards and buttons subtly rotate when hovered.
*   **Typography**: Uses Google Fonts `Patrick Hand` (Content) and `Permanent Marker` (Headings).
*   **Hard Shadows**: Solid, offset shadows to create depth without softness.

## Inspiration Sources
*   *Based on concepts from PaperCSS and DoodleCSS.*
*   *Technique*: CSS `border-radius` morphing (blob generator style).

## Usage
Simply link the `style.css` in your HTML:
```html
<link rel="stylesheet" href="style.css">
```
Ensure you include the Google Fonts:
```html
<link href="https://fonts.googleapis.com/css2?family=Patrick+Hand&family=Permanent+Marker&display=swap" rel="stylesheet">
```
